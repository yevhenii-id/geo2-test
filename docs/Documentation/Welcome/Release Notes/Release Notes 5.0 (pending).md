# New Features

|  **Summary**          |  **Description**                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
|:----------------------|:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Hub: Geo AI Assistant | **Geo2 Hub 5.0 introduces the new Geo AI Assistant**, built directly into Hub to help you plan and manage deliveries faster. You can create and update routes, assign drivers and vehicles, add stops to routes, and edit delivery details using simple text or voice commands. The new “Ask anything” field allows you to quickly execute actions or get instant answers about Geo2 features and workflows. The assistant can also navigate you to the correct page while keeping the dialog open, so you stay in context. It reduces manual work, minimizes errors, and helps you complete daily planning tasks in seconds. |
